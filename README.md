## Usage

follow this command
```bash
# Example for Alacritty:
# Make sure to create the target directory if it doesn't exist
mkdir -p ~/.config/alacritty
# Create the symlink
ln -s "$(pwd)/dotfiles/alacritty/alacritty.toml" ~/.config/alacritty/alacritty.toml```
```
