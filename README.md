# Добро пожаловать, командир.  
Rice config for **Hyprland** on Arch Linux,  
running on my **ROG Zephyrus G15** (_dionysus_). 

## Features
  - Animated **Neofetch**  
  - Dynamic **Waybar**  
  - ASCII **Cava Visualizer**  
  - Nord-inspired **neon-radioactive theme**
