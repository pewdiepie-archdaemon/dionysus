## Custom **Zsh** config
Shell configuration with aliases and environment variables for the Dionysus setup.

## Features
- Custom aliases for common commands
- Environment variables for tools
- Integration with the dotfiles ecosystem

## Usage
Copy to your home directory:
```bash
cp .zshrc ~/.zshrc
```

Make sure zsh is your default shell:
```bash
chsh -s /usr/bin/zsh
```
